# Yes-No-Maybe
This is another small program I created while on my journey of learning Python as I begin my path toward my first days in a computer science degree. This program asks the user to enter a question, and the program then returns a randomized answer of Yes, No, or Maybe.

I built this project to practice using user input, functions, loops, random selection, input validation, and conditional statements in Python.

How to use:
1. Open main.py in Python, PyCharm, VS Code, or any other Python IDE.
2. Run the program.
3. Input a question when prompted (make sure to include a ?).
4. Receive a randomized answer.

Type “quit” or “exit” if you want to close the program.
